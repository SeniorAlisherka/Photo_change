1) Предполагается, что pip и python установлены 
2) Загружаем библиотеки из requirements (pip3 install -r requirements.txt)
Можно так же при желании для этого создать venv.
3) Добавляем папку с фотографиями (в которой есть Список.xlsx и 
папка "Фото до обработки", в которой все изначальные фото).
В Список.xlsx обязательно должны быть колонки "Название файла после обработки"
и "Ссылка, содержащая название файла до обработки" на фото в диске
4) запускаем наш script.py 
5) Папка "Непонятные фото" - это фотки, которые выбиваются каким-либо
образом из Список.xlsx
Папка "Фото для ручной обработки" - это фотки, с которыми скрипт не справился,
их нужно самим обработать и засунуть в папку "Фото после обработки".
6) Готово, можно копировать папку "Фото самозапись" обратно в гуглдиск.
7) При повторном запуске скрипта, он все удалит и заново сделает
